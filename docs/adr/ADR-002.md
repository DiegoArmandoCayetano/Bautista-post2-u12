# ADR-002: Factory + Strategy para seleccion de procesador

## Estado
Aceptado — 2026-05-17

## Contexto

Existen diferentes tipos de pedido con algoritmos distintos.

## Decision

Se usa Strategy junto con Factory para seleccionar el procesador
adecuado en tiempo de ejecucion.

## Consecuencias

### Positivas

- Facil agregar nuevos tipos
- Menor acoplamiento

### Negativas

- Mayor numero de clases
- Mas abstraccion
