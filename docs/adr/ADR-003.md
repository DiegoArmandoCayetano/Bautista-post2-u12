# ADR-003: Spring Events para notificaciones

## Estado
Aceptado — 2026-05-17

## Contexto

El sistema necesita notificar a multiples canales
cuando un pedido es procesado.

## Decision

Se utiliza Spring Events mediante ApplicationEventPublisher
y listeners independientes.

## Consecuencias

### Positivas

- Bajo acoplamiento
- Escalable

### Negativas

- Debugging mas complejo
- Orden de ejecucion no garantizado
